# Agent Checklist

## Before Publishing
- [ ] Mission is one sentence
- [ ] Scope has responsibilities and non-responsibilities
- [ ] Inputs validated with formats and rules
- [ ] Outputs have success/failure conditions
- [ ] Constraints documented
- [ ] Dependencies include fallback strategy
- [ ] Tools follow least privilege
- [ ] Memory model has promotion criteria
- [ ] Failure taxonomy has detection/recovery/escalation
- [ ] Observability fields are defined
- [ ] Evaluation targets and review cadence set
- [ ] Version and changelog present

## Before Promotion to Production
- [ ] Tests exist
- [ ] Failure modes tested
- [ ] Documentation complete
- [ ] Security review done
- [ ] Health metrics configured
- [ ] Observability enabled
