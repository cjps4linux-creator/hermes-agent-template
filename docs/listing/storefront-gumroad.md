## Why this template

Most AI agents shipped today are prompt wrappers with no contract, no failure taxonomy, no input/output definitions, and no validation. Buyers stop saying *the agent hallucinated* and start validating behavior before shipping.

This template turns scattered prompts into auditable, testable, evaluatable systems.

---

## What you’ll be able to do

- Define agent identity, mission, scope, and non-responsibilities explicitly
- Specify inputs, outputs, constraints, dependencies, and tool permissions
- Model failure modes with detection, recovery, and escalation rules
- Run a Python validator against any agent contract before runtime
- Scaffold new agents in under 30 seconds
- Review agents against a checklist before publishing
- Migrate the pattern to your existing LLM backend or runtime
